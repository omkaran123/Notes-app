# Flutter Notes App

A simple end-to-end Flutter application created for learning and interview demonstration.

#App Description

This Notes App allows users to:
- View notes
- Add new notes
- Delete notes

The app demonstrates basic Flutter concepts such as widgets, state management, and navigation.
